# SmartOps

AI-powered customer follow-up tool for Nigerian food and fashion businesses.

## What it does
Tells small business owners exactly which customers to call today
before they lose them — delivered via WhatsApp.

## The problem it solves
Nigerian SMEs lose customers silently. No system tracks who stopped
buying. SmartOps fixes that with AI-powered churn detection.

## Built with
- Python
- Pandas
- scikit-learn (coming Month 2)
